*Управляющая
Системы, ориентированные на взаимодействие с реальным миром.Совершенно другое устройство исполнение.<br>
Произошло событие в реальном мире -> оцифровано -> что-то происходит. 
Особенности:
1. Встроенное исполнение
2. ограниченные вычислительные ресурсы
3. автономное исполнение
4. Работа в режиме реального времени
   * предсказуемо и работает в тот срок, в который все обговорено
   * != быстро
   * != точно 
   * != без отказов
